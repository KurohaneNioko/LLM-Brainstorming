# LLM Brainstorming
-------
Code for "Brainstorming Brings Power to Large Language Models of Knowledge Reasoning"

### Requirement
+ transformers >= 4.38
+ pytorch >= 2.4
+ flash_attn = 2.x

### Dataset

Datasets are from huggingface.

### Usage

Set the path of LLMs and run all cells. Prompts can be found in code.
